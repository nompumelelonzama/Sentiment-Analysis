

# 🎭 Sentient AI - Sentiment Analysis Platform

## 📌 Overview

Sentient AI is an advanced sentiment analysis platform that uses AI to interpret emotions in text data. Upload your CSV or Excel files and get instant sentiment analysis with beautiful visualizations.

## ✨ Features

- 🔍 **Real-time Sentiment Analysis** - Classifies text as Positive, Negative, or Neutral
- 📊 **Interactive Dashboard** - Visual charts including donut, bar, scatter, and heatmap
- 🌍 **Multi-language Support** - English, French, and Spanish interfaces
- 📄 **Executive Reports** - Auto-generated strategic recommendations
- 💾 **Export Capabilities** - Download results as CSV or text reports
- 📈 **Score Distribution** - Box plots showing sentiment intensity

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| Python | Backend logic |
| Streamlit | Web framework |
| NLTK VADER | Sentiment analysis engine |
| Plotly | Interactive visualizations |
| Pandas | Data processing |
